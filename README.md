# Book Management System in C
NIM: L03250XXX  
Nama: Adnan Aiman  

Program ini dibuat untuk mengelola data buku perpustakaan. Pengguna dapat memasukkan data beberapa buku, menyimpannya dalam file `.txt`, serta menampilkan data dalam bentuk tabel.

## Fitur:
- Input data buku
- Menyimpan ke file `data_buku.txt`
- Menampilkan tabel data
- Menggunakan struct dan file handling

## Catatan
Program ini dibuat sebagai bagian dari tugas / praktikum mata kuliah dasar pemograman.
